## 🎯 使用指南

### 1. 主页
- 查看所有账号（方块布局）
- 点击账号进入任务列表
- 点击"+"添加新账号
- 悬停账号卡片显示删除按钮

### 2. 账号详情页
- 查看该账号的所有任务
- 点击"新增任务"创建任务
- 运行或删除任务

### 3. 创建任务
- 填写任务名称和签到时间
- 选择 Chat 或手动输入 Chat ID
- 配置删除延迟和动作间隔
- 添加动作序列
- 点击"创建任务"

### 4. 导航
- 点击网站名称返回主页
- 使用面包屑导航
- 点击设置图标访问系统设置
- 点击 GitHub 图标访问项目主页
